# My Isaac Mod

## Overview
This mod enhances the gameplay experience in "The Binding of Isaac: Repentance" by introducing new features and improvements. 

## Installation
1. Download the mod files.
2. Place the `my-isaac-mod` folder into the `mods` directory of your game installation.
3. Launch the game and enable the mod from the mods menu.

## Features
- Custom gameplay mechanics that enhance player experience.
- Improved user interface elements.
- Various quality-of-life improvements.

## Files
- **main.lua**: The main entry point for the mod, handling initialization and core functionality.
- **metadata.xml**: Contains metadata about the mod, including name, version, and author.
- **content/gfx/ui/placeholder.txt**: Placeholder for future graphic assets related to the user interface.
- **resources/scripts/callbacks.lua**: Defines callback functions for interacting with game events.
- **resources/scripts/utils.lua**: Contains utility functions for common tasks within the mod.
